# Quickstart

This document provides the necessary steps to get the application running locally.

## Prerequisites

-   Node.js (version 20.x or higher)
-   npm (version 10.x or higher)

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/CodeForgeChris/heavy-dashboard.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd heavy-dashboard
    ```
3.  Install the dependencies:
    ```bash
    npm install
    ```

## Running the Application

To start the development server, run the following command:

```bash
npm run dev
```

The application will be available at [http://localhost:5173](http://localhost:5173) and will automatically open in your default browser.
